# Akagoria, the revenge of Kalista

Akagoria is a 2D single-player [role-playing game](http://en.wikipedia.org/wiki/Role-playing_video_game) in an [open world](http://en.wikipedia.org/wiki/Open_world) with a [top-down view](http://en.wikipedia.org/wiki/Top-down_perspective).
